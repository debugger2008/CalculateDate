# calculatedate

无非法字符检查输入机制，无日期正确与否检查机制，待有空增加输入二个日期后，求出相隔天数的功能。如果用API来解决，可以参考DateAdd(timeinterval,number,date)，或者JAVA里的LocalDate，Durati    on和Period。
